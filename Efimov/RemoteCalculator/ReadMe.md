# Задача №2. Удалённый калькулятор (Remote Calculator)

- При выполнении задачи, были использованы уже реализованные на занятиях классы-обертки `SocketWrapper` (`ServerSocket`, `ClientSocket`, `Socket`, `WSAInitializer`).
- Было реализовано корректное закрытие сокетов во всех ветках исполнения.
- Сделана обязательная обработка сетевых ошибок (`NetworkException`, недоступность сервера, обрыв соединения).
- Возможность запуска клиента и сервера на 127.0.0.1 (localhost).
- Порт сервера вынести в константу или конфигурацию (по умолчанию 12345).
- Код компилируется под `Windows` с поддержкой C++ 17 стандарта и выше.

---

### Задание 
Клиент отправляет арифметическое выражение, сервер вычисляет результат и возвращает его. Задача направлена на парсинг сетевых пакетов и обработку бизнес-логики на стороне сервера.

---

### Протокол обмена 
- Клиент → Сервер: `CALC <a> <op> <b>\n` (пример: `CALC 10.5 * 2`)
- Сервер → Клиент: `RESULT: <value>\n` или `ERROR: <reason>\n`

---

### Выполнено 
- Базовый уровень.
- Асинхронный уровень.
- Логирование.
- Обработка ошибок.

---

### Компиляция сервера

`g++ -std=c++17 -Wall -o server.exe base_socket.cpp core.cpp server_socket.cpp server_core.cpp server.cpp -lws2_32 -pthread`

---

### Компиляция клиента
`g++ -std=c++17 -Wall -o client.exe base_socket.cpp core.cpp client_socket.cpp client_core.cpp client.cpp -lws2_32`
