# Задача №1. Сервер точного времени (Time Sync)

- При выполнении задачи, были использованы уже реализованные на занятиях классы-обертки `SocketWrapper` (`ServerSocket`, `ClientSocket`, `Socket`, `WSAInitializer`).
- Было реализовано корректное закрытие сокетов во всех ветках исполнения.
- Сделана обязательная обработка сетевых ошибок (`NetworkException`, недоступность сервера, обрыв соединения).
- Возможность запуска клиента и сервера на 127.0.0.1 (localhost).
- Порт сервера вынести в константу или конфигурацию (по умолчанию 12345).
- Код компилируется под `Windows` с поддержкой C++ 17 стандарта и выше.

**Задание:**   
Клиент запрашивает у сервера текущие дату и время. Сервер возвращает отформатированную строку. Задача демонстрирует базовый запрос-ответ и работу с системным временем.

**Протокол обмена:**  
- Клиент -> Сервер: `GET_TIME\n`  
- Сервер -> Клиент: `TIME: YYYY-MM-DD HH:MM:SS\n` или `ERROR\n`

**Выполнено:**
- Базовый уровень.
- Асинхронный уровень.
- Логирование.
- Обработка ошибок.

**Компиляция программы**   
- (`g++ -std=c++17 -o client.exe .\base_socket.cpp .\core.cpp .\client_socket.cpp .\client_core.cpp .\client.cpp -lws2_32` Для клиента.)

- (`g++ -std=c++17 -o server.exe .\base_socket.cpp .\core.cpp .\server_socket.cpp .\server_core.cpp .\server.cpp -lws2_32 -pthread` Для сервера.)
